# Delta Force 战术数据分析工具

**实时游戏监控 | 战术地图分析 | 智能收益预测**

🎮 专为《三角洲行动》(Delta Force) 玩家打造的全方位数据分析平台

## ✨ 核心功能

### 💻 实时游戏监控
- 🖥️ **屏幕捕获** - 自动检测游戏画面
- 🎯 **智能识别** - OpenCV色彩检测金色/红色高价值物品
- 📊 **实时记录** - 自动保存出生点、收获物品、撤离价值
- 💾 **数据同步** - 即时上传到分析系统

### 🗺️ 战术地图系统
- 📍 **出生点分析** - 优势方/劣势方刷新点详解
- 💰 **高价值区域** - 热门搜刮点位标注
- 🚁 **撤离规划** - 多个撤离点风险评估
- 🛣️ **战术路线** - 推荐跑图路径和时间预估

### 📊 物资分析
- 📈 **出货概率** - 各地图物品掉落率统计
- 🎰 **单次模拟** - 模拟一局游戏的收益
- 📉 **批量统计** - 100次模拟的收益分布
- 🔥 **地图对比** - 热力图可视化最赚钱的地图

### 🤖 智能推荐
- 🎯 **装备推荐** - 基于你的战绩推荐最优配置
- 💡 **战术建议** - AI分析你的游戏习惯
- 📊 **收益预测** - 预测下一局的可能收益

## 🛠️ 技术栈

- **前端框架**: Streamlit 1.28+
- **数据处理**: Pandas, NumPy
- **可视化**: Plotly (交互式图表)
- **游戏监控**: OpenCV, MSS (屏幕捕获)
- **数据存储**: JSON, CSV

## 📊 功能模块

### 11个主要功能页面
1. 🏠 **战备配置** - 地图/模式选择，装备配置
2. 💰 **战备计算器** - 装备成本计算
3. 🎖️ **干员指南** - 干员技能和搭配
4. 🗺️ **战术地图** - 5标签页地图详解
5. 📊 **物资分析** - 4标签页数据分析
6. 🎒 **装备推荐** - 武器装备数据库
7. 📈 **数据管理** - 导入/导出/备份
8. 📋 **游戏记录** - 历史战局查询
9. 📉 **深度分析** - 高级统计图表

11. 💻 **实时监控** - 游戏画面监控

## 🎯 使用场景

### 游戏前
- 查看地图热力图，规划搜刮路线
- 计算装备成本，控制投资
- 查看历史数据，选择最赚钱的地图

### 游戏中
- 实时监控自动记录高价值物品
- 根据出生点查看推荐路线
- 评估撤离点风险

### 游戏后
- 自动保存战局数据
- 分析收益趋势
- 优化战术策略
