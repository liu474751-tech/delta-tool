# 🎮 三角洲工具 - 快速开始指南

## 📦 一键启动

双击 `start.bat` 即可启动！选择：
- 选项1：只启动桌面客户端（实时记录游戏）
- 选项2：只启动Web页面（数据分析）
- 选项3：同时启动两个模块（推荐）

## 🎯 核心功能

### 1️⃣ 实时记录（桌面客户端）

```
desktop/main.py
```

**功能：**
- 🎯 自动识别出生地
- 📦 实时记录收获物品
- 💰 计算撤出价值
- 📊 自动上传数据

**使用步骤：**
1. 启动客户端
2. 切换到"📊 实时会话"标签
3. 点击"🎮 开始新会话"
4. 开始游戏，自动识别
5. 结束时点击"✅成功撤离"或"❌阵亡"

### 2️⃣ 数据分析（Web页面）

```
streamlit run app.py
```

**功能：**
- 📊 实时会话监控
- 📈 历史数据分析
- 🗺️ 地图出货统计
- 💡 智能装备推荐

**页面导航：**
- 🏠 战备配置 - 装备成本计算
- 💻 实时会话 - 查看当前游戏
- 📋 游戏记录 - 历史数据统计
- 📉 深度分析 - 数据可视化

## 📁 数据存储

所有数据自动保存在：
```
C:\Users\你的用户名\Documents\DeltaTool\
```

包含：
- `live_session.json` - 实时会话
- `game_records.json` - 历史记录
- `game_records_export.csv` - CSV导出
- `stats.json` - 统计数据

## 🔧 首次安装

### 桌面客户端
```bash
cd desktop
install.bat
```

会自动安装：
- Python依赖
- OCR引擎
- PyQt6界面

### Web页面
```bash
pip install -r requirements.txt
```

## 💡 使用技巧

### 提高OCR识别率
1. 游戏画质：高/极高
2. 分辨率：1920x1080+
3. OCR引擎：PaddleOCR（推荐）

### 快捷键
- `F9` - 手动截图
- `F10` - 开始/停止监控
- `F11` - 立即识别
- `Ctrl+S` - 快速保存

### 实时查看
- Web页面每2秒自动刷新
- 点击"🔄 立即刷新"手动更新
- 侧边栏显示实时状态

## 📊 数据流程

```
游戏画面
    ↓ 截图
桌面客户端 OCR识别
    ↓ 保存
Documents/DeltaTool/
    ↓ 读取
Web页面显示
```

## ❓ 常见问题

**Q: 识别不准确？**
A: 使用PaddleOCR + 高画质 + 全屏游戏

**Q: Web数据不显示？**
A: 检查桌面客户端是否运行 + 是否开始会话

**Q: 如何导出数据？**
A: 在"📋 游戏记录"页面点击"📥 导出为CSV"

**Q: 程序崩溃怎么办？**
A: 数据已自动保存，重启程序即可恢复

## 🎮 推荐工作流

1. **游戏前**：启动桌面客户端，开始新会话
2. **游戏中**：自动识别记录，无需操作
3. **游戏后**：打开Web页面查看分析
4. **长期**：积累数据，查看趋势
